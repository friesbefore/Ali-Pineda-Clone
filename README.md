 # 🌐 Ali Pineda Website Clone

## 📌 Overview
This project is a front-end clone of the **Ali Pineda Creates** website. It was built to practice and improve my skills in HTML, CSS, and responsive web design.

The goal of this project is to replicate the layout, design, and overall user interface of the original website while gaining hands-on experience in modern web development techniques.

---

## 🚀 Features
- Responsive navigation bar
- Hero section with call-to-action buttons
- Featured work showcase
- Testimonial section with styled cards
- Contact section
- Footer with social media links
- Mobile responsive design using media queries

---

## 🛠️ Built With
- HTML5  
- CSS3  
- Google Fonts  
- Font Awesome  

---

## 📱 Responsive Design
The website is fully responsive and adapts to different screen sizes:
- Desktop  
- Tablet  
- Mobile
